<h1>IT Project: Creating an Azure Virtual Machine</h1>
This tutorial outlines the step by step implementation to launch a single Azure Virtual Machine
<br />


<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machines/Compute)
- Remote Desktop

<h2>Operating Systems Used </h2>

- Windows 10 (21H2)

<h2>High-Level Deployment and Configuration Steps</h2>

- Go to https://portal.azure.com/#home
- Log in and Confirm Active Azure subscription
- Select Create resource and Create Virtual Machine
- Configure Virtual Machine to Desired size for desired use
